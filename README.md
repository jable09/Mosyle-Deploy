# Mosyle-Deploy
This space exists to function as a repository for PKGs to live in order for Mosyle to install applications on managed MacBooks.

## Adding PKGs

- Within GitHub, locate the "Releases" section on the right side
- Click the latest release
- Click the pencil icon in the top right
- Scroll down to the section to "Attach binaries..."
- Locate the file and upload it
- Update release
- Locate the newly added asset and right click
- Copy link
- Within Mosyle, paste this link into an "Install PKG" "PKGs" package
